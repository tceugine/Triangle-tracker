# Triangle Tracker

### Description
A web-based application that determines the type of a triangle based on three side lengths provided by the user. It identifies whether the sides form an Equilateral, Isosceles, or Scalene triangle, or if they cannot form a triangle at all.

### Setup Instructions
1. Clone this repository to your local machine.
2. Ensure you have `index.html`, `style.css`, and `script.js` in the same directory.
3. Open `index.html` in any web browser to run the application.
4. No additional installations or server setups are required.

### How it Works

| Input: Side A, B, C | Behavior | Expected Output |
| :--- | :--- | :--- |
| 5, 5, 5 | All sides are equal | Equilateral Triangle |
| 5, 5, 8 | Two sides are equal | Isosceles Triangle |
| 3, 4, 5 | All sides are different | Scalene Triangle |
| 1, 2, 8 | Sum of two sides <= third | NOT a triangle |
| [Empty], 5, 5 | Missing input | Please enter valid numbers |
